# ROI Staff Contact Directory

A React Native mobile application built for **Red Opal Innovations (ROI)** as part of the TAFE assessment CI_MobileAppsDev_AE_Pro2.

## Overview

The app allows ROI staff to view, search, add, edit and delete employee contact records. It is designed for Android (portrait phone and landscape tablet) and follows the ROI Corporate Style Guide branding.

## Screens

| Screen | Description |
|--------|-------------|
| Staff List | Scrollable, searchable list of all staff members |
| Staff Profile | Full contact details for a selected staff member |
| Add / Edit Staff | Form to add a new member or edit an existing one |

## Tech stack

- React Native (JavaScript)
- Expo / Expo Snack
- React Navigation (stack)

## Run locally

```bash
git clone https://github.com/Yarik55/ROI-StaffDirectory.git
cd ROI-StaffDirectory
npm install
npx expo start
```

Scan the QR code with **Expo Go** on your Android device.
